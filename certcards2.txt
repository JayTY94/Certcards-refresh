26

Logical Block Addressing (LBA)
LBA is a linear numbering system for disk sectors, replacing older Cylinder/Head/Sector addressing. Each sector is assigned a unique number starting at 0.
Formula:
Offset (bytes) = LBA × Bytes per sector

Common sector sizes: 512 bytes (legacy) or 4096 bytes (4Kn).

Python
Q:
How can generators be used to implement a producer-consumer pattern?

A:
Generators can act as producers by yielding data, which consumers can iterate over, allowing for efficient and decoupled data processing.

Example:
def producer():
    for i in range(10):
        yield i

def consumer(gen):
    for item in gen:
        print(f"Consumed: {item}")

gen = producer()
consumer(gen)

Grammar
Dependent (Subordinate) Clause
A dependent clause contains a subject and a verb but cannot stand alone as a complete sentence. It depends on an independent clause to supply full meaning. Dependent clauses are often introduced by subordinating conjunctions such as because, although, while, if, or that. For example, “because the system failed” must attach to another clause to be grammatical.





71
Grammar
Relative Clause
A relative clause modifies a noun and provides additional information about it. It is typically introduced by a relative pronoun such as who, which, that, or whom. For example, in “The report that you sent was incomplete,” that you sent modifies report. Relative clauses function adjectivally and are embedded within noun phrases.





72
Grammar
Subordinate Clause
A subordinate clause is any dependent clause that is grammatically subordinate to another clause. This includes adverbial, relative, and noun clauses. Subordination signals hierarchy in sentence structure, indicating that one clause provides context or content for another rather than equal status.

limit the number of retries in a retry decorator by maintaining a counter within the decorator and exiting after reaching the maximum number of retries.

Example:
from functools import wraps

def retry(max_retries):
    def decorator(func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            attempts = 0
            while attempts < max_retries:
                try:
                    return func(*args, **kwargs)
                except Exception as e:
                    attempts += 1
                    print(f"Attempt {attempts} failed: {e}")
            raise Exception("Maximum retries exceeded")
        return wrapper
    return decorator
